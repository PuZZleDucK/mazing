
### Mazing

A JavaScript random maze generator and maze solver.


#### Maze solving algorithms

- Depth first
- Breadth first
- As the crow flies heuristic

#### Deployment

Deployed on [GH Pages](https://puzzleduck.github.io/mazing/index.html)
